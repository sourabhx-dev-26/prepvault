# PrepVault — GitHub Pages Prototype

This is a static education marketplace prototype for selling:

- PDFs
- Notes
- Current affairs PDFs
- Mock tests
- Quizzes

## How to host on GitHub Pages from mobile

1. Create a new GitHub repository.
2. Upload these files to the root of the repository:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. Open repository Settings.
4. Open Pages.
5. Source: Deploy from a branch.
6. Branch: `main`
7. Folder: `/root`
8. Save.
9. Wait until GitHub gives you the live link.
